\begin{abstract}
Este informe presenta el desarrollo de dos modelos del trabajo practico: un sistema de colas M/M/1 y un modelo de inventario con politica de revision continua $(Q,r)$. Para M/M/1 se documentan los supuestos teoricos, el diseno experimental, la simulacion por eventos discretos en Python y la comparacion contra resultados estacionarios cuando corresponde. Los casos de cola infinita con $\rho \geq 1$ se conservan como simulaciones de horizonte finito, pero no se comparan contra teoria estacionaria porque el modelo no posee regimen estable en esas condiciones. Para inventario se comparan tres politicas bajo demanda diaria Poisson, ventas perdidas y costos de orden, mantenimiento y faltante, incorporando una referencia esperada simplificada como punto de contraste. La comparacion con AnyLogic se integrara cuando esten disponibles los resultados del modelo desarrollado en paralelo.
\end{abstract}
