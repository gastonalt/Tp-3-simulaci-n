# Memoria del proyecto

Antes de ejecutar cualquier tarea en este proyecto, leer todos los archivos existentes dentro de la carpeta `docs/`.

La carpeta `docs/` funciona como memoria viva del TP: alli se registran decisiones, parametros, avances, resultados parciales, pendientes y convenciones de trabajo. Si una tarea modifica el rumbo del proyecto o produce informacion relevante, actualizar o crear el documento correspondiente en `docs/` antes de cerrar la tarea.
